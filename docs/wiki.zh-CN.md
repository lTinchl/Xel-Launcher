## 软件简介

Xel Launcher 是一个面向 Windows 的第三方游戏启动器，基于 AntdUI 构建。它用于管理《明日方舟》和《明日方舟：终末地》的 PC 客户端，支持多服务器入口、切服、游戏下载/更新、账号切换、启动参数、联动软件启动和签到。

当前支持的入口：

| 游戏 | 服务器 |
| --- | --- |
| 明日方舟 | 官服、Bilibili 服 |
| 明日方舟：终末地 | 官服、Bilibili 服、国际服、Google Play 服 |

## 准备工作

使用前请确认：

- 系统为 Windows。
- 已安装 Microsoft Edge WebView2 Runtime。内置浏览器和部分网页功能需要它。
- 已从项目 Releases 页面下载安装包或便携版。
- 如需切服，建议把 Xel Launcher 放在与游戏相同的磁盘分区，便于使用硬链接提升切服速度。

注意事项：

- 请勿把游戏根目录放在与某些反作弊组件限制冲突的位置，否则启动器可能无法读取或替换游戏文件。
- 切服会覆盖目标游戏目录中的部分服务器差异文件。执行前务必确认选中的是正确的游戏根目录。
- 启动器不会主动上传账号文件或登录凭据，账号切换和备份都在本地完成。

## 第一次启动

1. 打开 Xel Launcher。
2. 在左侧栏选择一个游戏或服务器入口。
3. 点击主按钮。如果本地未设置游戏路径，启动器会提示选择游戏根目录。
4. 选择包含游戏主程序的目录：
   - 明日方舟：目录中应包含 `Arknights.exe`。
   - 终末地：目录中应包含 `Endfield.exe`。
5. 路径校验通过后，启动器会保存配置，后续无需重复选择。

如果主按钮显示 `安装游戏`，说明启动器没有在当前路径检测到完整游戏文件，可以直接通过启动器下载并安装。

## 主界面说明

主界面主要由四个区域组成：

- 左侧栏：切换不同游戏和服务器入口，也可以添加或管理入口。
- 顶部栏：语言切换、主题切换、设置、帮助、关于、GitHub 和 Bilibili 链接。
- 中央区域：显示游戏封面、公告和相关信息。
- 底部操作区：账号管理、账号选择、开始/安装/更新按钮和更多菜单。

底部主按钮会根据状态自动变化：

- `安装游戏`：未检测到本地游戏。
- `更新游戏`：检测到可用更新。
- `开始游戏`：游戏已安装且无需更新。
- `暂停` / `继续`：下载或更新任务进行中。

## 配置游戏路径

### 首次启动时配置

1. 选择游戏入口。
2. 点击 `开始游戏` 或 `安装游戏`。
3. 按提示选择游戏根目录。
4. 启动器会检查目录中是否存在对应主程序。

### 在游戏设置中修改

1. 进入对应游戏页面。
2. 点击右下角更多菜单。
3. 选择 `游戏设置`。
4. 在 `游戏安装路径` 中点击 `更改路径`。
5. 选择正确的游戏根目录。

`游戏设置` 中还提供 `打开文件目录`，可快速打开当前配置路径。

## 安装和更新游戏

Xel Launcher 会检测本地版本和远端版本，并根据结果切换主按钮状态。

安装或更新流程：

1. 选择游戏入口。
2. 当主按钮显示 `安装游戏` 或 `更新游戏` 时点击它。
3. 首次安装时选择安装目录；已有路径时会使用当前配置路径。
4. 等待下载、校验、解压、安装或清理流程完成。
5. 完成后主按钮会恢复为 `开始游戏`。

如果下载或更新中断，可以再次点击 `继续`。遇到缓存异常时，可在更多菜单中使用 `清理下载缓存` 后重试。

## 启动游戏

1. 选择要启动的游戏或服务器入口。
2. 如已启用账号切换，在账号下拉框中选择账号。
3. 点击 `开始游戏`。
4. 启动器会按需要执行以下步骤：
   - 结束可能占用文件的游戏进程。
   - 切换服务器差异文件。
   - 恢复所选账号数据。
   - 附加自定义启动参数。
   - 启动联动软件。
   - 启动游戏本体。

启动成功后会显示提示。如果全局设置中开启了 `启动游戏后关闭软件` 或 `启动游戏后隐藏至托盘`，启动器会在确认游戏启动后自动关闭或隐藏。

## 服务器切换

当多个服务器入口共用同一个游戏目录时，启动器会在启动前自动切换对应服务器文件。

支持的切换方向：

- 明日方舟官服与 Bilibili 服互转。
- 终末地官服、Bilibili 服、国际服、Google Play 服之间切换。

推荐配置方式：

1. 先为官服入口设置正确的游戏路径。
2. 在官服的 `游戏设置` 中使用 `同步路径到 Bilibili服 / 国际服 / GooglePlay服`。
3. 确认多个入口指向同一个游戏根目录。
4. 启动不同入口时，启动器会自动部署对应服务器文件。

手动替换：

- 明日方舟 B 服入口提供 `将文件替换为B服`。
- 终末地 B 服入口提供 `将文件替换为B服`。
- 终末地国际服入口提供 `将文件替换为国际服`。
- 终末地 Google Play 服入口提供对应替换功能。

手动替换会覆盖当前目录中的相关文件，执行前请确认路径无误，并确保游戏已退出。

## 账号管理

账号切换适用于：

- 明日方舟官服
- 终末地官服
- 终末地国际服

Bilibili 服和 Google Play 服入口不会显示账号切换开关。

### 启用账号切换

1. 进入对应游戏页面。
2. 打开右下角更多菜单。
3. 选择 `游戏设置`。
4. 开启 `启用账号切换`。
5. 返回游戏页后，底部会显示账号按钮和账号下拉框。

### 添加并保存账号

1. 点击底部账号按钮，进入账号管理。
2. 点击 `+ 添加账号`。
3. 输入账号名称。
4. 打开游戏并登录到这个账号。
5. 回到账号管理窗口，点击该账号的 `保存账号`。

保存后，启动器会把当前账号数据备份到本地账号目录。

### 切换账号

1. 在游戏页账号下拉框中选择账号。
2. 点击 `开始游戏`。
3. 启动器会在启动前恢复该账号数据。

账号管理窗口支持 `保存账号`、`设为默认`、`重命名`、`启用/禁用` 和 `删除`。删除账号需要二次确认。

## 自定义启动参数

每个游戏入口都可以独立配置启动参数。

1. 进入对应入口的 `游戏设置`。
2. 开启 `自定义启动参数`。
3. 在输入框中填写参数。
4. 下次启动该入口时会自动附加这些参数。

终末地 Google Play 服还提供 `Session Token` 配置项，可手动填写，也可使用 `自动获取 Token`。自动获取前需要确认游戏进程已经启动。

## 联动软件

联动软件用于在启动游戏时自动启动其他程序，例如辅助工具、统计工具或脚本。

1. 进入对应入口的 `游戏设置`。
2. 开启 `自定义联动软件`。
3. 点击 `管理联动软件`。
4. 点击 `+ 添加`，选择要启动的程序。
5. 如有需要，为该程序填写启动参数。

启用后，每次通过该入口启动游戏时，启动器会同时启动已配置的联动软件。

## 签到功能

右下角更多菜单提供 `签到` 入口，当前包含：

- 森空岛签到
- SKPORT 签到

### 森空岛签到

森空岛签到支持多种获取 Token 的方式：

- 扫码登录
- 手机验证码登录
- 账号密码登录
- 手动填写 Token

多个 Token 使用英文分号 `;` 分隔，支持多账号签到。配置完成后可以点击 `立即签到`。也可以开启：

- `启动器运行时签到`：启动器运行期间每日自动签到一次，并在右下角提示结果。
- `开机自动签到`：开机后后台执行一次签到，不显示主窗口，完成后通过系统通知提示。

### SKPORT 签到

SKPORT 签到支持：

- 账号密码登录获取 Token
- 手动填写 Token
- 多账号 Token，用英文分号 `;` 分隔
- 立即签到
- 启动器运行时签到
- 开机自动签到

如果签到失败，可先查看签到窗口中的日志，再确认网络、账号密码、Token 是否有效。

## 小工具和外部链接

游戏页左下角工具按钮会根据当前游戏提供常用入口：

- PRTS Wiki
- 明日方舟工具箱
- 明日方舟一图流
- 终末地一图流
- Warfarin Wiki
- Endfield Tools

默认会使用内置浏览器打开。若在设置中启用 `使用外部浏览器`，则会改用系统默认浏览器打开。

## 全局设置

点击顶部设置按钮进入全局设置。常用选项包括：

- `最小化到托盘`：关闭主窗口时最小化到系统托盘。
- `开机自启动`：随 Windows 启动。
- `启动游戏后关闭软件`：游戏启动成功后退出启动器。
- `启动游戏后隐藏至托盘`：游戏启动成功后隐藏到托盘，游戏退出后恢复。
- `使用外部浏览器`：网页链接使用系统默认浏览器打开。
- `使用硬链接切服`：优先使用硬链接替换服务器文件，提高切服速度。

设置窗口还包含：

- `日志`：查看启动器运行日志。
- `软件更新`：检查新版本，下载更新安装包或便携版。

顶部栏还提供：

- 语言切换：中文 / English。
- 主题切换：跟随系统 / 浅色 / 深色。

## 配置和数据位置

启动器配置默认保存于：

```text
%LocalAppData%\XelLauncher\config.json
```

等价路径通常为：

```text
C:\Users\<用户名>\AppData\Local\XelLauncher\config.json
```

账号备份目录：

```text
%LocalAppData%\XelLauncher\AccountBackups
%LocalAppData%\XelLauncher\EndAccountBackups
%LocalAppData%\XelLauncher\GlobalEndAccountBackups
```

如果需要重置启动器配置，请先关闭软件，再删除 `config.json`。如果需要重置账号备份，请删除对应备份目录。

## 常见问题

### 点击开始游戏时提示未找到 exe

通常是选择的目录不是游戏根目录。请重新选择包含 `Arknights.exe` 或 `Endfield.exe` 的目录。

### 切服很慢

通常是硬链接不可用，启动器回退到了文件复制。建议：

- 把 Xel Launcher 放到与游戏相同的磁盘分区。
- 在设置中启用 `使用硬链接切服`。
- 确认磁盘文件系统支持硬链接。

### 切服失败或提示文件被占用

请先退出游戏和相关进程，再重试。必要时重启电脑后再次执行切服。

### 账号切换失败

请确认：

- 已在游戏设置中启用账号切换。
- 已点击 `保存账号` 记录过该账号。
- 启动器有权限读取和写入账号文件。
- 游戏当前没有运行。

### Google Play 服无法正常带 Token 启动

请进入终末地 Google Play 服的 `游戏设置`，确认 `Session Token` 已填写。若使用 `自动获取 Token`，请先启动游戏并保持游戏进程运行。

### 内置网页打不开

请确认已安装 WebView2 Runtime。也可以在全局设置中启用 `使用外部浏览器`，改用系统默认浏览器打开网页。

### 下载或更新失败

可以尝试：

- 检查网络连接。
- 关闭代理或切换网络后重试。
- 使用更多菜单中的 `清理下载缓存`。
- 到项目 Releases 页面手动下载安装包或便携版。

### 启动后没有自动隐藏或关闭

请检查全局设置中的 `启动游戏后关闭软件` 和 `启动游戏后隐藏至托盘`。这些操作只有在启动器检测到游戏进程启动成功后才会执行。

## 使用建议

- 首次配置后，分别启动一次各服务器入口，确认路径、切服和账号状态正常。
- 手动替换服务器文件前，确认目标路径无误，并关闭游戏。
- 账号切换前，先为当前账号执行一次 `保存账号`。
- 重要账号数据建议定期备份 `%LocalAppData%\XelLauncher`。
